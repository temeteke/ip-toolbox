# IP Toolbox

PWA対応のネットワーク計算ツール - IPアドレス分析とCIDR計算

## 機能

### 1. ネットワーク情報の一覧表示

CIDR表記（例: `192.168.0.0/24`）を入力すると、以下の情報を計算します：

- ネットワークアドレス
- ブロードキャストアドレス
- ホスト範囲（最初のホスト〜最後のホスト）
- ホスト数
- サブネットマスク
- ワイルドカードマスク

### 2. 2進数表示

IPアドレスとサブネットマスクを2進数で表示します。

例：`192.168.0.0/24`
```
IPアドレス: 11000000.10101000.00000000.00000000
サブネットマスク: 11111111.11111111.11111111.00000000
```

### 3. IP判定機能

個別のIPアドレスが指定したネットワークに属するかを判定し、以下の情報を表示します：

- ネットワーク所属の判定（属する/属さない）
- IPの種類（ネットワークアドレス/ブロードキャスト/ホスト）

## 技術スタック

- **フレームワーク**: Nuxt 3
- **言語**: TypeScript
- **UIフレームワーク**: Vue 3
- **スタイリング**: Tailwind CSS
- **PWA**: @vite-pwa/nuxt

## セットアップ

### 依存パッケージのインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

### ビルド

静的サイトを生成：

```bash
npm run generate
```

ビルドされたファイルは `.output/public` に出力されます。

## デプロイ

このプロジェクトはGitHub Pagesに自動デプロイされます。

### GitHub Pages設定

1. リポジトリの Settings > Pages を開く
2. Source を「GitHub Actions」に設定
3. `main` または `claude/*` ブランチにプッシュすると自動デプロイされます

## PWA機能

このアプリケーションはPWA（Progressive Web App）に対応しており、以下の機能が利用できます：

- オフライン動作
- ホーム画面への追加（モバイル）
- 高速な読み込み
